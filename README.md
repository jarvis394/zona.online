# zona-online

ZONA for real men.

### Installing locally

```
git clone https://github.com/jarvis394/zona.online.git
cd zona.online
npm install
npm start
```

### Configuring

`process.env.TOKEN` = VK group's API token

`process.env.ID` = VK group's ID
